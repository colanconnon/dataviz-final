Justin Verlander (JV) MLB Pitch Data Visualization
Summary

This visualization uses data from Justin Verlander (JV) a MLB ( Major League Baseball ) pitcher.

The main purpose of this visualization is to walk through Justin Verlander's lifetime pitching data and break down

  • What types of pitches he throws
  • What are his dominant pitches
  • Visualize strike out data by pitch type and location

Dataset

The dataset used for this project is the MLB statcast dataset.
All data was downloaded in python and exported to csv using the pybaseball package

Special Terms

ERA: The number of runs given up per 9 innings pitched.

About This Visualization

This visualization is an interactive slide show, where each scene displays a graph and as you progress throw the slide show it goes into more detail. Navigation is in the bottom right hand corner, and you can go forward, backwards, and on some slides (2, 4) you can even go down and up.

Each scene has a graph and some text explaining some insight from the graph. Transitioning from scene to scene will display a new graph.

For annotations each graph has some text explaining the graph, and the ERA vs Fastball graph has an annotation explaining a key point in the graph.

Visualization parameters are used in the last two graphs. The Strikeouts per Year graph allows you to go forward and backward in time to see the strikeouts over the years and this adjusts the year parameter. The Individual Pitch Analysis allows you to adjust the year, event type and batter stance to drill down into the data even more.

The main triggers are mouse over events to show tool tips for details on demand. In the individual pitch visualization, the trigger applies opacity to all other points in the scatter plot and makes the point the user is focusing on bigger. The button click on the strikeout visualization slide also changes the year parameter as the user progresses through the visualization, so the user can see how strikeout pitches change over time.

User interface Events:

All Charts expect Fastball vs ERA over time support mouse over events on data items to show tooltips

The strikeouts over time has a button click event which changes strikeout data over time. This changes the year parameter over time.

The individual pitch analysis allows users to select three parameters from a dropdown: Year, Game Event, and Batter Stance.
